# J.A.R.V.I.S — Голосовой ИИ-ассистент

> \\\*"Just A Rather Very Intelligent System"\\\*  
> Голосовой ассистент на Python с интеграцией qwen 2.5, распознаванием речи на русском языке и синтезом голоса.

\---

## Архитектура

```
┌─────────────────────────────────────────────────────────┐
│                        JARVIS                           │
│                                                         │
│  Микрофон → STTEngine ──► CommandHandler ──► TTSEngine  │
│                │              │                         │
│                │         ┌────┴─────┐                   │
│                │         │Classifier│                   │
│                │         └────┬─────┘                   │
│                │              │                         │
│                │    ┌─────────┼──────────┐              │
│                │    ▼         ▼          ▼              │
│                │  Local    Launch      LLMClient        │
│                │  cmds      App       (Claude API)      │
│                │                                        │
│  ◄─────────────┴──────────── speak() ──────────────────│
└─────────────────────────────────────────────────────────┘
```

|Модуль|Библиотека|Роль|
|-|-|-|
|`STTEngine`|`SpeechRecognition` + Google|Распознавание речи с микрофона|
|`TTSEngine`|`pyttsx3`|Синтез речи в отдельном потоке|
|`LLMClient`|`anthropic`|Сложные запросы → Claude|
|`CommandHandler`|—|Маршрутизация команд|
|Terminal UI|`colorama`|Цветной вывод в консоль|

\---

## Требования к системе

* **Python** 3.10 или выше
* **Микрофон** (USB, встроенный или Bluetooth)
* **ollama с установленной нейросетью qwen2.5:7b**

### Дополнительные системные пакеты

**Ubuntu / Debian:**

```bash
sudo apt-get update
sudo apt-get install -y python3-dev portaudio19-dev python3-pyaudio espeak-ng
```

**macOS:**

```bash
brew install portaudio espeak
```

**Windows:**

* PyAudio устанавливается через pip автоматически
* Голоса SAPI доступны из коробки (панель управления → Специальные возможности → Синтез речи)

## Использование

### Слово активации

Скажите **«Джарвис»** перед каждой командой.

```
«Джарвис, который час?»
«Джарвис, открой Chrome»
«Джарвис, что такое квантовая запутанность?»
«Джарвис, пока»
```

### Поддерживаемые команды

|Фраза|Действие|
|-|-|
|`Джарвис, который час`|Сообщает текущее время|
|`Джарвис, какое сегодня число`|Сообщает дату|
|`Джарвис, как ты`|Статус системы|
|`Джарвис, открой Chrome / Telegram / VS Code`|Запускает приложение|
|`Джарвис, запусти Firefox / Spotify / Discord`|Запускает приложение|
|`Джарвис, спасибо`|Получает ответную любезность|
|`Джарвис, стоп / пока / выключись`|Завершает работу|
|**Любой другой вопрос**|Передаётся в Claude API|

## Конфигурация (изменения в `jarvis.py`)

```python
# Язык распознавания
LISTEN\\\_LANGUAGE = "ru-RU"        # Варианты: "en-US", "uk-UA", "de-DE"

# Чувствительность микрофона (меньше = чувствительнее)
ENERGY\\\_THRESHOLD = 300

# Скорость речи (слов в минуту)
TTS\\\_RATE = 165

# Максимальная длина диалога с LLM (в парах вопрос-ответ)
MAX\\\_CONTEXT\\\_TURNS = 10

# Модель Claude
CLAUDE\\\_MODEL = "claude-sonnet-4-5"
```

### Добавление нового приложения

В `jarvis.py` найдите `APP\\\_MAP` и добавьте запись:

```python
(("ключевое слово", "другое слово"),  "Имя приложения",
    \\\["команда-запуска"] if sys.platform != "win32" else \\\["команда-для-windows"]),
```

Пример:

```python
(("фотошоп", "photoshop"),  "Adobe Photoshop",
    \\\["photoshop"] if sys.platform != "win32" else \\\[r"C:\\\\Program Files\\\\Adobe\\\\Photoshop\\\\Photoshop.exe"]),
```

## Устранение неполадок

|Проблема|Решение|
|-|-|
|`ModuleNotFoundError: pyaudio`|`sudo apt-get install portaudio19-dev \\\&\\\& pip install pyaudio`|
|Микрофон не найден|Проверьте системные настройки звука; запустите `python -c "import speech\\\_recognition as sr; print(sr.Microphone.list\\\_microphone\\\_names())"`|
|Речь не распознаётся|Говорите чётче; увеличьте громкость; уменьшите `ENERGY\\\_THRESHOLD`|
|`AuthenticationError`|Проверьте `ANTHROPIC\\\_API\\\_KEY`|
|Нет русского голоса TTS|Установите русский голосовой пакет ОС; на Linux: `sudo apt-get install espeak-ng-data`|
|Таймаут API|Проверьте интернет; увеличьте `LLM\\\_TIMEOUT`|

### Список доступных микрофонов

```python
python -c "import speech\\\_recognition as sr; print('\\\\n'.join(f'{i}: {n}' for i, n in enumerate(sr.Microphone.list\\\_microphone\\\_names())))"
```

Затем укажите нужный индекс в `STTEngine.\\\_init\\\_microphone()`:

```python
self.\\\_mic = sr.Microphone(device\\\_index=2)  # укажите нужный индекс
```
